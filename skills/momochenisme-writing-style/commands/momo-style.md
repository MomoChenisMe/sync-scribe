# MomoChenIsMe 寫作風格

啟用 MomoChenIsMe 的獨特寫作風格來撰寫文章。

## 使用方式

```
/momo-style              → 以 MomoChenIsMe 風格撰寫（通用）
/momo-style tutorial     → 技術教學風格
/momo-style product      → 產品介紹風格
/momo-style review       → 年度回顧風格
```

## 指令說明

根據用戶指定的文章類型，套用對應的寫作風格：

### 通用風格（無參數或不認識的參數）
套用 MomoChenIsMe 的核心寫作原則：
- 對話式、親切的語氣
- 自嘲幽默（使用 XD、（苦笑）等）
- 謙虛誠實，坦承不足
- 實作導向，強調「怎麼做」

### 技術教學風格 (`tutorial`)
適用於程式教學、技術文章：
1. **開場**：描述問題背景，引起讀者共鳴
2. **本文**：
   - 環境/前置需求
   - 步驟 1、步驟 2...（搭配截圖）
   - 💡 補充說明或 FAQ
3. **結尾**：總結成果，鼓勵讀者嘗試

### 產品介紹風格 (`product`)
適用於工具推薦、產品業配：
1. **開場**：為什麼需要這個工具？
2. **本文**：
   - 🚀 核心功能
   - 📘 實際使用範例
   - 🔑 優缺點（誠實列出）
3. **結尾**：🎯 適合哪些人使用？

### 年度回顧風格 (`review`)
適用於年度總結、個人反思：
1. **開場**：設定時間背景
2. **本文**：
   - 發生的事件（時間軸）
   - 學到的教訓
   - 感謝的人
3. **結尾**：對未來的期望

## 寫作特色重點

### 語氣風格
- 像跟朋友聊天，不是在講課
- 多用「大家應該都知道...」、「我得坦白說...」開頭
- 結尾常用「以上就是今天的分享，希望對大家有幫助！」

### 符號使用
| 符號 | 用途 |
|-----|------|
| 💡 | 重要提示、注意事項 |
| ✨ | 亮點、精彩內容 |
| XD | 幽默、自嘲 |
| 「」| 專有名詞、書名 |
| **粗體** | 強調關鍵字 |

### 程式碼區塊模式
1. 先解釋這段程式碼要做什麼
2. 放上程式碼區塊
3. 逐行解釋重點
4. 💡 補充常見問題或替代方案
