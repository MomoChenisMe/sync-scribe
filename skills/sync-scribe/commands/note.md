# Note 筆記管理

快速管理你的筆記。

## 使用方式

```
/note <內容>           → 添加筆記
/note list            → 列出所有筆記
/note search <關鍵字>  → 搜尋筆記
/note delete <標題>    → 刪除筆記
```

## 指令說明

根據用戶輸入的參數執行對應操作：

### 添加筆記（預設行為）
當參數不是 `list`、`search`、`delete` 開頭時，視為新增筆記：
1. 讀取工作目錄下的 `NOTES.md`
2. 自動判斷分類：
   - 內容涉及工作/程式/專案 → `工作`
   - 內容涉及生活/美食/旅遊 → `生活`
   - 其他 → `未分類`
3. 從內容摘要生成標題
4. 在對應分類區塊下新增筆記（格式：`## [YYYY-MM-DD] 標題`）
5. 儲存檔案並確認新增成功

### 列出筆記 (`list`)
1. 讀取 `NOTES.md`
2. 顯示所有筆記，按分類整理
3. 顯示標題、日期和內容預覽

### 搜尋筆記 (`search <關鍵字>`)
1. 讀取 `NOTES.md`
2. 搜尋標題或內容包含關鍵字的筆記
3. 顯示匹配的結果

### 刪除筆記 (`delete <標題>`)
1. 讀取 `NOTES.md`
2. 找到匹配標題的筆記（支援模糊匹配）
3. 刪除該筆記
4. 儲存檔案並確認刪除

## 範例

```
/note 捷運站附近的咖啡廳很好喝
→ ✅ 已新增筆記：捷運站咖啡廳 [分類：生活]

/note list
→ 📝 筆記列表：

  【工作】
  - [2025-02-05] API 金鑰位置

  【生活】
  - [2025-02-05] 捷運站咖啡廳

/note search 咖啡
→ 🔍 搜尋結果：
   - [2025-02-05] 捷運站咖啡廳
     內容：捷運站附近的咖啡廳很好喝

/note delete 捷運站咖啡廳
→ 🗑️ 已刪除筆記：捷運站咖啡廳
```

## 檔案格式

`NOTES.md` 檔案格式如下：

```markdown
# 筆記

## 分類：工作

### [2025-02-05] API 金鑰位置
API 金鑰存放在環境變數 `API_KEY` 中。

## 分類：生活

### [2025-02-05] 捷運站咖啡廳
捷運站附近的咖啡廳很好喝，推薦拿鐵。

## 分類：未分類

（無筆記）
```
