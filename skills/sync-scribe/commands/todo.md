# Todo 待辦事項管理

快速管理你的待辦事項清單。

## 使用方式

```
/todo <任務描述>        → 添加待辦
/todo list             → 列出所有待辦
/todo done <任務描述>   → 標記完成
/todo delete <任務描述> → 刪除待辦
```

## 指令說明

根據用戶輸入的參數執行對應操作：

### 添加待辦（預設行為）
當參數不是 `list`、`done`、`delete` 開頭時，視為新增待辦：
1. 讀取工作目錄下的 `TODOS.md`
2. 在 `# 待辦事項 (Active Todos)` 區塊下新增 `- [ ] <任務描述>`
3. 儲存檔案並確認新增成功

### 列出待辦 (`list`)
1. 讀取 `TODOS.md`
2. 顯示所有待辦事項，區分未完成和已完成
3. 以清楚的格式呈現

### 完成待辦 (`done <任務描述>`)
1. 讀取 `TODOS.md`
2. 找到匹配的待辦項目（支援模糊匹配）
3. 將 `- [ ]` 改為 `- [x]`
4. 將該項目移至 `# 已完成 (Completed)` 區塊
5. 儲存檔案並確認完成

### 刪除待辦 (`delete <任務描述>`)
1. 讀取 `TODOS.md`
2. 找到匹配的待辦項目（支援模糊匹配）
3. 刪除該項目
4. 儲存檔案並確認刪除

## 範例

```
/todo 買牛奶
→ ✅ 已新增待辦：買牛奶

/todo list
→ 📋 待辦事項：
   - [ ] 買牛奶
   - [ ] 寫報告

/todo done 買牛奶
→ ✅ 已完成：買牛奶

/todo delete 寫報告
→ 🗑️ 已刪除：寫報告
```

## 檔案格式

`TODOS.md` 檔案格式如下：

```markdown
# 待辦事項 (Active Todos)

- [ ] 待辦項目 1
- [ ] 待辦項目 2

# 已完成 (Completed)

- [x] 已完成項目 1
```
